Резюме для руководства

Тревел-индустрия переживает крупнейший сдвиг дистрибуции со времён появления OTA. Model Context Protocol (MCP) от Anthropic стал де-факто стандартом подключения AI-агентов к тревел-инфраструктуре: за 18 месяцев его приняли Sabre, Expedia, Booking.com, Turkish Airlines и ещё более десятка компаний. Каждый крупный OTA уже эксплуатирует AI-агента — некоторые обрабатывают более 50% обращений автономно. Бизнес-эффект измерим: рост конверсии до 15%, снижение стоимости обслуживания на 65%, скачки трафика до 200%. На российском рынке ни одна платформа не запустила официальный MCP-сервер — у Tutu.ru есть реальное окно первопроходца.

Раздел 01

14 тревел-компаний уже запустили MCP-серверы

С момента выпуска Anthropic протокола MCP в ноябре 2024 года темп внедрения в тревеле оказался одним из самых высоких среди отраслей. К апрелю 2026 года как минимум 14 компаний эксплуатируют продакшен- или бета-MCP-серверы — и список растёт каждую неделю.

GDS-провайдер

Sabre Corporation

23 сентября 2025 — запуск агентных API на базе проприетарного MCP-сервера, интегрированного в облачную платформу SabreMosaic. MCP работает как «универсальный переводчик» для AI-агентов: поиск, бронирование и обслуживание перелётов, отелей, постбукинг-процессы. Под капотом — 50+ петабайт исторических и реальных данных (совместно с Google).

OTA → ChatGPT App

Expedia Group

Открыл исходный код MCP-сервера тревел-рекомендаций на GitHub. 6 октября 2025 вошёл в первые 7 пилотных приложений ChatGPT — наравне с Booking.com. Apps SDK самого OpenAI построен поверх MCP.

Авиакомпания

Turkish Airlines

Один из первых авиа-MCP-серверов: mcp.turkishtechlab.com. Поиск рейсов, статус в реальном времени, PNR, регистрация, багаж, программа лояльности Miles&Smiles. Доступен из Claude.ai, Claude Desktop и VS Code Copilot.

NDC-оркестрация

TPConnects

Архитектурно самая значимая реализация: MCP-слой, нормализующий несовместимые версии NDC (с 18.2 до 24.4+) в единый AI-интерфейс. Решает одну из главных проблем интероперабельности в авиа-дистрибуции. ConvertEngine: −60% объёма поисковых запросов, 97% точность.

OTA

lastminute.com / Kiwi.com / Booking.com

lastminute.com развернул MCP-сервер для авиаконтента в рамках 3-летней AI-дорожной карты. Kiwi.com запустил прототип MCP для поиска рейсов в августе 2025. Booking.com — одновременно с Expedia интегрировался в ChatGPT.

Отели / Активности

Apaleo, TourRadar, Peek

Apaleo — первый PMS с MCP: 237 API-эндпоинтов → MCP-инструменты (сентябрь 2025). TourRadar подключил 50 000+ многодневных туров к AI-платформам. Peek открыл инвентарь экскурсий и впечатлений.

Open-source экосистема: на GitHub существуют community-MCP-серверы для Amadeus, Duffel, Skyscanner, Airbnb, TripAdvisor, Китайских ж/д (12306) и Швейцарских ж/д. Для Aviasales API уже есть сторонний MCP-сервер (maratsarbasov/flights-mcp) — контент Aviasales доступен AI-агентам, хотя сама компания его не создавала.

Hopper: кейс полной автономии

HTS Assist (октябрь 2025) — самый продвинутый пример полностью автономного тревел-агента. Система бронирует рейсы, отменяет поездки, оформляет возвраты и управляет сбоями без участия человека, обучившись на 16 миллионах диалогов. При этом 90% выручки Hopper ($850M) теперь приходит от B2B-лицензирования этой AI-технологии авиакомпаниям (Virgin Australia, Frontier) и финансовым институтам.

McKinsey SkyLink: внутренний кейс

McKinsey развернула AI-агента SkyLink для корпоративных командировок в марте 2025. Результаты: 30 000+ бронирований через агента, время бронирования сократилось с 10 минут до менее 2, 95% положительных оценок. Уже 20% бронирований в США и Канаде проходят через SkyLink.

Раздел 04

Архитектура: MCP как стандарт подключения

Пользователь → AI-агент (Claude / ChatGPT / Gemini) → MCP Client → MCP Server → Travel API (GDS / NDC / OTA)

Протокол MCP решает фундаментальную проблему тревел-дистрибуции: фрагментацию API-интерфейсов. NDC существует в нескольких несовместимых версиях, GDS используют проприетарные форматы, каждый отель и авиакомпания имеет свой API. MCP выступает единым слоем, через который AI-агент может взаимодействовать с любым поставщиком.

Аналитик Джонни Торсен (Serko) сравнил роль MCP с HTTP на заре интернета. Skift предупредил в декабре 2025: компании, нечитаемые для AI-агентов, станут невидимыми в новой модели дистрибуции.

Платёжный слой — последний нерешённый элемент

Большинство AI-агентов пока перенаправляют на сайт OTA для оплаты. Но Visa в апреле 2026 запустила Intelligent Commerce Connect — первая крупная платёжная сеть, формально открывшая инфраструктуру для агентной коммерции: machine-to-machine-платежи и токены аутентификации для AI-агентов.

Рынок AI-технологий в тревеле растёт с $3.4B (2024) до $13.9B к 2030 — при CAGR 26.7%. Доля AI в инвестициях в тревел-стартапы подскочила с 10% (2023) до 45% (H1 2025). Глобальный рынок тревела генерирует $1.67 трлн в год, только OTA — $94B выручки.

ChatGPT с 800M WAU уже сделал тревел одной из первых вертикалей приложений. Google AI Mode обслуживает 75M DAU и интегрирует бронирование с Booking.com, Expedia, Marriott. Perplexity AI партнёрится с Tripadvisor и SelfBook для отельных бронирований.
