План внедрения
Пошаговый маршрут без лишней теории
Ориентир: 15–25 минут
Шаг 01

Установить iTerm2

Это нормальная замена стандартному Terminal: удобнее вкладки, профили и общая эргономика. Ставь через Homebrew, а если его нет — скачай приложение с официального сайта.

brew install --cask iterm2или iterm2.com
Шаг 02

Поставить Oh My Zsh

Это базовый слой для настройки zsh. После установки shell уже станет приятнее, а дальше на него удобно навешивать тему и плагины.

Oh My Zshzsh framework
Шаг 03

Подключить Powerlevel10k

Самая заметная часть визуального апгрейда. После перезапуска iTerm2 откроется мастер, где достаточно подтвердить иконки, выбрать вид prompt и включить git-статус.

Powerlevel10kprompt themewizard on restart
Шаг 04

Добавить случайный цвет фона

Не обязательный, но очень живой штрих: при каждом запуске iTerm2 фон будет слегка меняться. Важно вставить функцию в самое начало ~/.zshrc, до конфигурации Oh My Zsh.

optionalfun tweak
Шаг 05

Поставить плагины подсказок и подсветки

zsh-autosuggestions показывает продолжение команды серым текстом, а zsh-syntax-highlighting подсвечивает корректность ввода. Вместе это самая полезная часть setup-а.

autosuggestionssyntax highlighting
Шаг 06

Настроить ~/.zshrc

Включи нужные плагины, добавь локаль для русского языка и перечитай конфиг через source ~/.zshrc. Это финализирует рабочее окружение.

plugins=(...)LANG / LC_ALL
Шаг 07

Включить системные уведомления для Claude Code

Если гоняешь несколько сессий параллельно, iTerm2 умеет сообщать, когда агент ждёт ввода. Путь: Profiles → Terminal, затем включить Silence bell и настроить Notification center alerts.

Claude CodeiTerm2 notifications
Шаг 08

Запомнить горячие клавиши вкладок

Для многозадачности это важнее, чем кажется: новые вкладки, переход по номерам и прыжки между соседними сессиями экономят море микродвижений.

Что дают

Первый плагин показывает серые подсказки из истории, второй — делает команды визуально «правильными» или «подозрительными» ещё до запуска.

Фрагмент plugins=(...)~/.zshrc

  git
  macos
  docker
  zsh-autosuggestions
  zsh-syntax-highlighting
)

source ~/.zshrc
Случайный цвет терминалавставить в начало ~/.zshrc
PRELINE="\r\033[A"

function color {
    echo -e "\033]6;1;bg;red;brightness$((1 + $RANDOM % 255))\a"$PRELINE
    echo -e "\033]6;1;bg;green;brightness$((1 + $RANDOM % 255))\a"$PRELINE
    echo -e "\033]6;1;bg;blue;brightness$((1 + $RANDOM % 255))\a"$PRELINE
}

Практика
Как лучше пройти setup
1
Сначала ставь основу

iTerm2 → Oh My Zsh → Powerlevel10k. Только потом плагины и косметические фишки.

2
Редактируй ~/.zshrc осознанно

Блок случайного цвета должен быть в самом начале файла, иначе Oh My Zsh может перехватить поток и эффект сломается.

3
После каждого смыслового шага проверяй визуально

Перезапусти iTerm2, посмотри на prompt, проверь подсказки и только потом добавляй следующее.

4
Если работаешь с агентами — докрути уведомления

Именно этот шаг превращает красивый терминал в реально удобную рабочую станцию для нескольких CLI-сессий.
